# Week5 UX最適化レビュー（Big4向け）

## 1. 3秒把握
- Headerで「何のデモか」を明示
- Phase Stepperで進行位置を即時表示
- Side Panelで評価対象者・アラート・次アクションを固定表示

## 2. 1分説明
- 各タブ先頭に「経営向け要約（3行）」を配置
- 分析タブでKPI5種、監査整合、根拠ドリルダウンを一画面化
- AI判定基準タブで「AIは最終決裁しない」を明示

## 3. 5分提案
- 自動レポートで改善アクション（担当/期限/期待効果）を提示
- 監査タブで例外解除の理由・承認・履歴を追跡可能
- 乖離案件をドリルダウンして改善論点を提示可能

## 4. 表現ルール反映
- 用語補助: KPIと監査項目に短文の意味説明を添付
- 色規約: `#2e2e38` / `#ffe600` を中心に過剰警告を抑制
- ガバナンス: lock/edit可否をUI disabledで明確化

## 5. 残課題
- レポートPDFファイル出力は未実装（画面内生成）
- 複数評価対象者を横比較する経営ビューは次フェーズで追加余地
